# CODTECH Internship Documentation

This repository contains all four internship tasks completed during the CODTECH Full Stack Development internship.

The goal across all tasks was to build practical, real-world projects using frontend and backend technologies, while keeping the code clean, understandable, and easy to run.

## Task 1: API Integration

In this task, I built a weather dashboard that fetches live weather details from external APIs.

What was done:

1. Created a search-based weather interface for any city.
2. Integrated geocoding and weather APIs to fetch live data.
3. Displayed current weather, hourly forecast, and weekly forecast.
4. Added responsive UI and proper loading/error handling.

## Task 2: Chat Application

In this task, I built a real-time chat app where multiple users can communicate instantly.

What was done:

1. Built backend with Node.js and Express.
2. Implemented real-time messaging using Socket.IO.
3. Added username support, active user list, and recent message history.
4. Included basic rate limiting and reconnection handling for stability.

## Task 3: Real-Time Collaborative Document Editor

In this task, I built a collaborative editor where multiple users can edit the same document in real time.

What was done:

1. Built frontend using React for a dynamic and responsive UI.
2. Built backend using Node.js and Express.
3. Used Socket.IO rooms for real-time collaboration.
4. Connected MongoDB for document persistence and autosave support.
5. Added clean project structure, environment config, and clear documentation.
